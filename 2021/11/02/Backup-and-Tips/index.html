将项目部署到github

github方面

  1. 创建一个新的 repository
  2. 记下仓库地址

本地方面
使用 Git Bash Here

  1. cd 到java项目目录下
  2. git init 初始化在当前项目的目录中生成本地的git管理
  3. git add . 将项目上所有的文件添加到仓库中, 如果要添加特定文件, 只需要把 . 换成特定的文件名即可
  4. git commit -m "first commit" 表示对此次提交的注释, 双引号里面的内容即为注释内容
  5. git remote add origin https://xxxx.git 自己的仓库地址
  6. git push -u origin main 把代码上传到github仓库

注: 出现错误：fatal:refusing to merge unrelated histories, 意思是拒绝合并不相关的历史，这是因为在Github创建仓库时，我默认创建了README.md文件，而本地是没有的，所以出现冲突了，这时候，我们需要在第六条命令之前，输入 git pull origin main --allow-unrelated-histories , 然后再执行第六条命令

而IDEA自带有git工具，也可以使用IDEA的git工具进行push.
